Projeto: Sistema de Avaliação de Atletas
Descrição

Este projeto tem como objetivo criar uma aplicação em JavaScript capaz de receber informações de um atleta, calcular e exibir dados importantes como categoria, IMC e média válida das notas dadas pelos jurados.

Funcionalidades

Cálculo da categoria do atleta com base na idade:

Infantil: 9 a 11 anos

Juvenil: 12 a 13 anos

Intermediário: 14 a 15 anos

Adulto: 16 a 30 anos

Sem categoria: demais idades

Cálculo do IMC (Índice de Massa Corporal):
IMC = peso / (altura * altura)

Cálculo da média válida das notas, descartando a maior e a menor antes de calcular a média.
